Ajuda

As ações com este símbolo só podem ser executadas por usuários administradores

1) Como posso adicionar um tanque novo?

Para criar um tanque, abra o menu clicando no canto superior esquerdo, com o menu aberto clique na opção " Criar Tanque", no formulário digite apenas o nome do Tanque e deixe o recurso de localização do seu celular/tablet habilitado, e clique no botão "Criar Tanque"

2) Como consulto os tanques que criei?

Para consultar seus tanques cadastrados, Bata clicar na opção do menu " Gerenciar Tanques", Para localizar os tanques, primeiramente ative o recurso de localização do seu dispositivo, depois abra o menu clicando no canto superior esquerdo e clique na opção " Localizar tanques", será aberto um mapa com marcações das localizações dos seus tanques.

3) Como cadastro um novo funcionário?

Para cadastrar um novo funcionário abra o menu clicando no canto superior esquerdo, com o menu aberto clique na opção " Adicionar Funcionário", após isso preencha os dados do formulário. Note que, ao digitar o CEP, os dados de endereço devem ser preenchidos automáticamente. Ao final do preenchimento basta clicar no botão " Criar Funcionário".

4) Como enviar avisos?

Para enviar um aviso abra o menu clicando no canto superior esquerdo, e clique na opção " Enviar Aviso". No campo "Título", digitar o assunto do aviso, e no campo "Aviso" digitar a mensagem que se deseja enviar e no botão "Mandar aviso".
